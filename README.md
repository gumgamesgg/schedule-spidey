# ScheduleSpidey
This mod changes (nearly) all npc skins to a spiderman skin and removes the eyes and accessories.

**⚠️CAUTION:** This mod currently only works for the `alternate` Version of the game! 
Make sure you are using the correct branch on steam.



---

## 🛠 Requirements

Before installing, make sure you have the following versions:
* **[MelonLoader v0.7.1](https://github.com/LavaGang/MelonLoader/releases/tag/v0.7.1)** (or newer)
* **Schedule I v0.4.4 Alternate**

## 📥 Installation Guide

Follow these steps to get Spidey into your game:

1. **Install MelonLoader:** Make sure it is correctly set up in your Game folder.
2. **Download ScheduleSpidey:** Download the latest ZIP file from the Releases section or Nexus.
3. **Extract:** Unzip the downloaded folder on your computer.
4. Open your **Schedule I** game directory.
5. Move the contents of the extracted `Mods` folder into the `Mods` folder created by MelonLoader.
